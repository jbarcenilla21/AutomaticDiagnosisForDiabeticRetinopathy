## Local Structure After Extraction

```
data/
├── images/         # 3500 retinal fundus images (.jpg), named 00000.jpg → 03499.jpg
├── train.csv       # 2000 samples  (id, eye, label)
├── val.csv         # 500 samples   (id, eye, label)
└── test.csv        # 1000 samples  (id, eye, -1)

outputs/            # submission CSVs generated by the notebook (1000×1 DR scores)
```

> Both `data/` and `outputs/` are excluded from git. Each team member must download and extract the dataset locally.

## CSV Format

Each row: `image_id, eye_indicator, label`

| Field | Values |
|-------|--------|
| `id` | 5-digit zero-padded integer — maps to `data/images/<id>.jpg` |
| `eye` | `0` = left eye, `1` = right eye (mirror right-eye images for consistency) |
| `label` | 0–4 for train/val; `-1` (unavailable) for test |
